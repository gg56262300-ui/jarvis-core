JARVIS OPS SHORTCUTS

1) Kiire üldseis:
npm run ops:lite

2) Request + confirm:
npm run confirm:one -- health
npm run confirm:one -- control_summary_compact

3) Request + confirm + plain output clipboardi:
npm run confirm:plain -- control_summary_compact
npm run confirm:plain -- execution_state_compact
npm run confirm:plain -- health

MÄRKUS:
control_summary_compact võib näidata hetkeks:
- terminalStage=confirmed
- terminalStatus=running
- pendingStatus=pending

See on hetke snapshot confirm-tsükli seest, mitte viga.
